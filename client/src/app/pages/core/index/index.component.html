Welcome to EVE Track!


EVE Track can be best described as an EVE Online API viewer, it enables you to view your EVE Online character information without having to log into the game.


My goal is to provide you with a simple-to-use API viewer while still providing a lot of information about your character(s).


For optimal use of this website, I encourage you to create an account. This will enable you to save multiple characters and quickly access their data. Don't worry, it's safe!
The source code of this website is publicly available on GitHub, so feel free to nose around and discover how it all works.
