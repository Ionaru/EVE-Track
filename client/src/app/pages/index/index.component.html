Welcome to EVE Track!

EVE-Track can be best described as an EVE Online API viewer, it enables you to view your EVE Online data without having to log into the game.

Our goal is to provide you with a simple-to-use API viewer while still providing a lot of information about your character.


For optimal use of this website, I encourage you to create an account and add API keys to that account. This way you do not have to enter the key every time you want to view the data. However it is possible to view data without creating an account, using the 'temporary key' section.
