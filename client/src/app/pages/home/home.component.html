Welcome to EVIE!


EVIE can be best described as an EVE Online API viewer, it enables you to view your EVE Online character information without having to log into the game.


My goal is to provide you with a simple-to-use API viewer that gives you as much useful information about your characters as possible.


To use this website you'll need to register an account. I've done this because I don't want my backend API to be publicly available just yet. In the future this will change. This will also enable you to save multiple characters and quickly access their data.
The source code of this website is publicly available on GitHub, so feel free to nose around and discover how it all works.


Currently functional pages:

  • Ore prices table
  • Character dashboard
  • Skills page
  • Wallet balance & journal
